The '''_SCREENX''' function returns the current program window's upper left corner column position on the desktop.


{{PageSyntax}}
:: positionX&amp; = _SCREENX


* Returns the current column pixel coordinate of a program window on the desktop.
* Use [[_SCREENIMAGE]] to find the current user's Windows desktop resolution to adjust the position with [[_SCREENMOVE]].
